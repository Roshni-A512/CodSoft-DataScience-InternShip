# CodSoft Data Science InternShip Portfolio

## About This Repository
This repository contains the machine learning projects I completed during my **CodSoft Data Science InternShip**. Through these projects, I gained practical experience in data preprocessing, exploratory data analysis, visualization, machine learning model development, and model evaluation using Python.
Each notebook demonstrates the complete workflow, from understanding the dataset to training a model and interpreting the results.

## Projects Included

### 1.Iris Flower Classification

**Objective**
Develop a machine learning model that classifies Iris flowers into one of three species using their sepal and petal measurements.
**Project Highlights**
* Loaded and explored the Iris dataset
* Performed data inspection and preprocessing
* Visualized the dataset using different plots
* Trained a classification model
* Evaluated model performance using accuracy

**Skills Applied**
* Data Analysis
* Data Visualization
* Classification
* Model Evaluation
